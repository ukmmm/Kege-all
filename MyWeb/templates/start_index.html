подготовка к Компьютерному ЕГЭ

10-11 классов

Существует множество сайтов для подготовки к КЕГЭ, такие как kpolyakov, kompege.ru, РешуЕГЭ, но ни один из этих сайтов не предоставляет возможность просмотра решения задач с помощью кодов. Поэтому, наша команда, в составе из 3 человек, решила написать данный сайт, с помощью которого школьники не только узнают какие задачи решаются с помощью языков программирования, но и смогут посмотреть решения данных задач на коде от нас и от других учеников.
